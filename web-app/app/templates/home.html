AI Meme Generator

Emojifier
About the Project

Emojifier is a project that adds Emojis to text. This is done using TFiDF vectorization and a logistic regression model. The model was trained on twitter data and only predicts about 15 emojis.


Meme Classifier
About the Project

Meme Classifier is a project that given a text, classifies it into one of 10 meme formats. This is done using TFiDF vectorization and a logistic regression model. The model was trained on data obtained from imgflip.
